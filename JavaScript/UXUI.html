Ex) 스타벅스 매장 위치에 매출 규모를 지도 아이콘의 크기로 표현매출을 상위 25%, 50%, 75% 를 기준으로 4개의 그룹으로 나누는 기준을 매출 평균으로 분석 하 지도에 표시하는 프로젝트를 한다. 분석방법론, 시각화, 매출을 어떻게 구간으로 (월마다, 주마다, 일) 분석 전략 / 위도 경도, 일매출, 매장의 이름, 직영점 여부, 매장의 그룹화(프리미엄 매장 여부) 진성데이터를 어떻게 수집할 것인가 ?(국가단위, 기업에서 고용 사용을 인정한 데이터, 민간 기업이지만 데이터 공유) ‣ 서비스 및 컨텐츠 전략 : 단말기 ( Smarter Platform, UX, User Centric, Networked Resourc) : 인지적 커뮤니케이션 (기술 중심에서 인간 중심, 입체적 의사 소통, 저장매체에서 검색과 실시간 정보화, 글로벌화) : Business Model (어플리케이션 레이어의 개방, 소프트웨어 플랫폼 중심, 수평적 결합, 창의력과 혁식) : 시스템 커뮤니케이션 ( 개별 요소에 어플리케이션화, 멀티미디어 Multi dimensional, 콘텐츠의 즉시적 제공, Multi Channel (Wifi, LTE))
