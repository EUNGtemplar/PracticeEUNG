에밋(Emmet) 표현법

!사용

괄호 만들기 : 태그명 tab

반복태그 만들기 : 태그 * n

포함관계 만들기 : 태그>태그>태그*n> 탭

조합 예제 : div>ul*2>li*2

조합 예제 : div>(ol>li*2)+(ul>li*2)

컨테츠 표현 : div{div content}

컨텐츠 반복표현 : div*3{content}

컨텐츠 반복표현($=숫자 리스트) : div*3{내용$}

id를 사용하는 태그 : div#id1

id를 활용하는 태그 : div#id$*3, div#id${content}*3

class를 활용하는 태그 : div.c$*3{content}
